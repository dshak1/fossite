# Homework Tracker

A simple Spring Boot web application to track homework uploads and earn badges.

## Features
- User Accounts (Simple username login)
- Profile customization (Choose animal avatar and color theme)
- Homework Upload (Camera or File)
- Badges (Daily Double, Streaks)
- History of uploads

## Prerequisites
- Java 17 or higher
- Maven (optional, wrapper included/standard maven works)

## How to Run

1. Open a terminal in the project root.
2. Run the application:
   ```bash
   mvn spring-boot:run
   ```
3. Open your browser to `http://localhost:8080`.

## Usage
1. Enter a username to login (e.g., "student1").
2. Go to **Settings** to choose your avatar.
3. Upload a homework picture.
4. Check your badges!
